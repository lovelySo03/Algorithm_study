Ch04 구현 

< EX4-1 >
어제 수업시간에 방향배열에 대해서 배우고, 잘 익혔기 때문에 이 부분은 그다지 어렵지 않았다. 방향배열을 배우기 전에는 if, for문을 쓰면서 하나하나 다 직접 세야되서 힘들었는데 방향배열을 배우고 나니까 체계적이고 간결하게 좌표를 이동시킬 수 있어서 좋았다. 다만, x,y 좌표를 어떻게 지정하는지에 따라서 정답이 달라질 수 있기 때문에, 책에서 좌표를 어떻게 지정했는지 자세히 보고 그것에 맞게 코드를 짜는 능력이 필요한 것 같다. 

< 왕실의 나이트 >
이 부분도 그다지 어렵지 않았다. 종이에 좌표를 하나하나 적으면서 하니까, 도움이 되었다.

< 게임 개발 >
이 문제의 경우에는 기존에 있던 방향배열에, 회전을 추가해서 조금 어렵게 느껴졌다. 또, 바다와 같이 갈 수 없는 구역을 설정해두고, 방문했던 장소를 다시 가는 것이 불가능하다는 것도 난이도를 높이는 하나의 요소로 작용했다. 일단은 책에 있는 예제코드를 보면서 설계방법을 배운 후에, 내 코드로 다시 구현해보는 연습을 하려고 한다.
